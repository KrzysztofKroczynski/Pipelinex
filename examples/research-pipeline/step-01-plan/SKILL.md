# Plan research queries

Read the research topic from `input/topic.txt`.

Generate exactly 5 search queries that together cover the topic from different angles:
1. Core definition and fundamentals
2. Recent developments or news (last 1-2 years)
3. Practical applications or use cases
4. Challenges, limitations, or criticisms
5. Comparisons with alternatives or competing approaches

Each query should be a short, focused search string — the kind you'd type into a search engine.

Save to state:
- `topic`: the full text of the topic file
- `queries`: list of 5 query strings

No other work needed here. The next step will run all queries.
